# IPL Cricket Analytics Dashboard | 2008–2024

Interactive Power BI dashboard analyzing 1,095 IPL matches 
across 17 seasons (2008–2024).

## Key Insights
- Mumbai Indians are the most successful team with 145+ wins
- Eden Gardens hosted the most IPL matches
- Teams strongly prefer to field after winning the toss
- IPL has grown consistently from 2008 to 2024
- Standard Class is the most used shipping mode

## Visuals Included
- Most Successful Teams (bar chart)
- Matches Per Season (line chart)
- Toss Decision Impact (bar chart)
- Top Venues by Matches (bar chart)
- Season slicer and Team slicer for interactive filtering

## Tools Used
- Microsoft Power BI Desktop
- Dataset: IPL Complete Dataset (Kaggle)

## Dataset
- 1,095 IPL matches
- 17 seasons (2008–2024)
- Columns: match id, season, city, date, teams, 
  toss winner, toss decision, winner, venue, 
  player of match

## How to View
1. Download the .pbix file
2. Open in Power BI Desktop (free)
3. Use slicers to filter by season or team
